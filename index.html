Pyrado’s Documentation¶

Pyrado is a framework for reinforcement learning tailored domain randomization based on PyTorch and numpy. The implementations are focus on modularity rather than on performance.

Installation¶

Please see the root-level readme file of SimuRLacra.

Where to start?¶

Check out the provided examples or run some of the existing scripts in Pyrado/scripts.
